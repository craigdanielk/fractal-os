

# FractalOS Phase Timeline (Gantt Overview)

Total Duration: 12–14 weeks  
Daily Usability Delivered: Days 7–12  
Architecture Finalisation: Weeks 10–12  
Hardening: Weeks 12–14

## Phase 0 — Minimum Viable FractalOS (MVF)
Timeline: 7–12 days  
Purpose: Earliest usable slice for daily operations.

### Deliverables
- Unified task manager  
- Unified time tracking  
- Clients → Projects → Tasks chaining  
- Daily & Weekly Cockpit views  
- Basic manifests  
- Drizzle base schema  
- Supabase foundation  
- CRUD endpoints  
- Personal + R17-compatible pipeline
